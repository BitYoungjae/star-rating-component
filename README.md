# 별점 컴포넌트

## 사용법

```tsx
<StarRating
  now={45} // 현재값
  max={100} // 최대값
  maximumStars={10} // 별의 최대 갯수, Optional, 기본값 5
  sStarColor='red' // 별 색깔, Optional, 기본값 red
  sBackgroundColor='#ddd' // 별의 배경색, Optional, 기본값 #ddd
/>
```
